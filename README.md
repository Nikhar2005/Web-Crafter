# Web-Crafter : Drag-and-Drop Website Builder 
WebCrafter is a simple drag-and-drop web design tool built with HTML, CSS, JavaScript, and Bootstrap. It allows users to create web pages effortlessly by dragging elements like text, images, buttons, and forms onto a customizable canvas.
